# Changelog - Grunt 怪物开发

## [Unreleased]

### Added
- 蓝图函数库调试工具：封装 `Quick Print` 节点，支持结构化格式输出
- 怪物工具函数：`SetFacingDirection()` 和 `GetFacingDirection()`（2D Sprite 翻转）
- 感知攻击来源函数：`GetHitDirection()`，预留击飞/击退/击倒接口

### Changed
- 伤害处理机制：修复 GE 未设置伤害值导致对怪物无效的问题
- 受击动画逻辑：改用 GE 受击保护，保护期间免疫伤害，结束后触发 `TryPlayHurtAnimation()`

### Fixed
- 怪物 Grunt 受伤硬直动画无法被打断的问题（通过受击保护机制重写）
